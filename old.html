Learn JavaScript

Variables

The code below creates two variables. Think of a variable as a container that holds something. Instead of holding food or clothes, variables hold data. Just as there are different types of containers, there are different types of variables. The three most basic or primitive types in JavaScript are: integers (numbers), strings (characters), and booleans (true/false). When you create a variable and assign it a value, it's called instantiating it.

// Instantiate two int variables var height = 5; var width = 5; // Create a string variable // You can use "" or '' but they have to match var colorOne = "black"; var colorTwo = 'red';

A more advanced type of JavaScript variables are something called objects. An object is a copy of a class, where a class is a collection of variables, and something else that you will learn later called functions. When you create a copy, or instance, of a class, you can use that variable to access the variables and functions of that object.

// Create a Mosaic object, which is an instance of the Mosaic class var moz = new Mosaic(height, width);

Calling Functions

Functions are lines of code that have been grouped together and can be called upon by a name. Functions can be attached to an object, or just for general use. When you call a function, you sometimes pass parameters to the function. There are pre-defined inputs that the function expects, you give them by putting them in parenthesis after the function name.

// console is an object that is automatically created by JavaScript // we can print to the console window by calling console.log console.log("Hello, World!"); // In terms of the Mosaic object the main function that you will use is setTileColor var moz = new Mosaic(5, 5); moz.setTileColor(0, 0, "black"); // With setTileColor you must provide an int for x, an int for y, and a // string for color, these are called parameters var colorOne = "red"; moz.setTileColor(0, 1, colorOne); // Parameters are passed in the parenthesis separated by commas, they can // be either variable names or an actual value, it does not matter moz.setTileColor(0, 2, "#000000");

Conditionals

Conditionals are statement that check conditions, which can be either true or false, and then do something depending on that state. Sometimes they are referred to as If-Then, Else-Then statements. If you want to check multiple conditions after each other, you can use else if. If, else if, and else statements are evaluated in sequential order, so you must be careful. If you have an else or an else if, you must have an if before it, but if you have an if, you do not need anything additional.

var moz = new Mosaic(5, 5); // Get a random number between 1 and 3 var random = 1 + Math.floor(Math.random() * 3); // If random equals 1, do this if (random == 1) { moz.setTileColor(0, 0, "purple"); } // If it isn't 1, check if its 2 else if (random == 2) { moz.setTileColor(0, 0, "yellow"); } // If its none of the above, do this else { moz.setTileColor(0, 0, "red"); }

Using Loops

The loop header has the 1. variable initialization, 2. the condition, and 3. the variable mutation.

  1. Variable initialization only happens at the first iteration (a.k.a run through) of the loop. It sets up our counter.
  2. The condition is checked before every iteration to check if the loop should run or not.
  3. The variable is then mutated, you can do any math operation you want to on the variable.
var moz = new Mosaic(5, 5); // This loops goes from 0 - 4 for (var x = 0; x < moz.width; x++) { moz.setTileColor(x, 0, "blue"); }

You can also put loops inside of other loops, this is called nesting. The inner loops finish first, then goes outwards.

var moz = new Mosaic(5, 5); // This loops goes from 0 - 4 for (var x = 0; x < moz.width; x++) { // This loop goes from 0 - 4 for (var y = 0; y < moz.height; y++) { moz.setTileColor(x, y, "yellow"); } }

As previously written, functions are lines of code that have been grouped together and can be called upon by a name. These functions can take inputs called parameters, and can return outputs.

You can have two functions with the same name, as long as they have different parameters, or orders of parameters. This is called function overloading.

var moz = new Mosaic(5, 5); function drawLine() { for (var x = 0; x < moz.width; x++) { moz.setTileColor(x, 0, "blue"); } } // We can return any value we want with functions, provided that it is a singular value function getSecretColor() { var secretColor = "#89692f"; return secretColor; } // You have a function name then in the () you have parameters function drawLine(y) { for (var x = 0; x < moz.width; x++) { moz.setTileColor(x, y, "blue"); } }

Mosaic Class Methods
  • Mosaic(height, width): Creates a new Mosaic object.
    • Example: var moz = new Mosaic(5, 5);
  • setTileColor(x, y, color): Sets a tiles's color value.
    • You can use it with HTML color name: moz.setTileColor(0, 0, "#gold");
    • Or a color hex code: moz.setTileColor(0, 0, "#9b4495");
  • setTileGradient(x, y, colorOne, colorTwo): Sets a tiles's color value as gradient.
    • Example: moz.setTileGradient(3, 4, "blue", "yellow");
  • setTileBorderColor(x, y): Sets a tiles's border color value.
    • Example: moz.setTileBorderColor(3, 4, "blue");
  • setTileBorderWidth(x, y, width): Sets a tiles's border width value.
    • Example: moz.setTileBorderWidth(3, 4, "3px");
    • Example: moz.setTileBorderWidth(3, 4, "thick");
    • Example: moz.setTileBorderWidth(3, 4, "medium");
  • setTileBorderStyle(x, y, style): Sets a tiles's border style value.
    • Example: moz.setTileBorderColor(3, 4, "dotted");
    • Example: moz.setTileBorderColor(3, 4, "dashed");
    • Example: moz.setTileBorderColor(3, 4, "double");
    • Example: moz.setTileBorderColor(3, 4, "solid");
  • setTileBorder(x, y, color, width, style): Sets a tiles's border.
    • Example: moz.setTileBorderColor(3, 4, "blue");
  • setTileText(x, y, text): Sets a tiles's inner text.
    • Example: moz.setTileText(0, 0, "o");
  • setTileOnClick(x, y, func): Set a click event function for a tile.
      Example: moz.setTileOnClick(3, 4, function() { console.log("Hello, World!"); });
  • setTileOnMouseOver(x, y, func): Set a mouse over event function for a tile.
      Example: moz.setTileOnMouseOver(3, 4, function() { console.log("I love Mosaic!"); });
  • getTileColor(x, y): Gets a tile's color value.
    • Example: var color = moz.getTileColor(3, 4);
  • clear(): Clears all pixel color values in a Mosaic.
    • Example: moz.clear();
  • Mosaic.height: Gets Mosaic height.
    • Example: var height = moz.height;
  • Mosaic.width: Gets Mosaic width.
    • Example: var width = moz.width;
Color Class Methods
  • Color.random(): Get a random color value.
    • Example: moz.setTileColor(0, 0, Color.random());
  • Color.invert(): Inverts a given color value.
    • Example: Color.invert("black");
    • Example: Color.invert("#3954AE");
    • Example: Color.invert("rgb(3, 24, 244)");
Other Methods
  • await sleep(ms): Pauses excecution for a certain amount of milliseconds.
    • Example: await sleep(500);
