# mosaic
Learn JavaScript with instaneous visual feedback while making computer-generated art.

Created by [Rees Draminski](https://github.com/reesdraminski).

## About
The addon, lib, and mode folders are all apart of CodeMirror, and were not written by the creator of this project.

api.js is where the Mosaic methods are located that allow the user to create the Mosaic art.

editor.js is where the CodeMirror and code execution mechanics are located.
